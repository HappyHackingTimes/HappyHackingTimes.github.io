前置き

実プロジェクトではIntelliJをメインに開発を進めることになると思います。

基本操作はVSCodeなど他のIDEと変わることはありませんが、大きな利点として充実したショートカット機能が挙げられます。

ここではそのショートカットについて何を実際に使っているか？を使用シーンをGifで示しながら列挙します。
（随時更新します‥ﾀﾌﾞﾝ）

これらのショートカットを使いこなすことで開発スピードが上がること間違いなし！
また、VSCodeでも拡張機能があるみたい（IntelliJ IDEA Keybindings）です。
ぜひ活用してみてください。

覚えよ。さらば救われん

とは言いつつもショートカットをすべて最初から覚える必要はありません。
何ができるかを把握さえすればいくつかの手段で該当の操作が行えます。

intelliJのプラグインの活用：Key Promoter X

https://plugins.jetbrains.com/plugin/9792-key-promoter-x

ショートカットを使わずに操作すると親切にポップアップされる（画面右下）プラグインです。
ウザかったらショートカットを覚えるんだ。

ショートカットの候補をリスト表示：control + T

表示されるリファクタリングリストから変数の変更を選択しています。

ショートカットをキーワード検索：command + shift + A

検索ワードは日本語も一応、使えます。

プロジェクトに関わるすべての検索：shift 2回連続で押す

gifではファイル名を検索して表示してますが、他にもあらゆる検索が可能です。

独断と偏見のショートカット集

経験的に使うシーンが多いものを列挙しています。
順番に意味はありません。使えると思ったものを覚えてとにかく使いましょう！

変数名を変更する：shift + F6

該当変数をファイルをまたいで変更してくれます。

選択文字列に合致する位置にカーソルをあてる：control + G

カーソルを指定方向に増やす：option 押したあとに option + ↑/↓

新規ファイル作成：command + N

エクスプローラーにフォーカス/表示・非表示：command + 1

フォーカスした状態では上下キーが効くのでそのまま対象ファイルを開くことも可能です。

変数呼び出し元に遷移：command + B

変数呼び出し先を検索：option + F7

変数は複数箇所から呼び出されるのが常なのでツリー状でどこからかを示してくれます。

呼び出しの大元から検索できるときはcommand + Bで可能（こっちの方が楽かも）

コードの折りたたみ/展開：command + shift + -/+

option + →/←で動く分だけ削除：option + delete

ファイル全体のコード整形：option + command + L

コードの移動：option + shift + ↑/↓

予め範囲選択しておけば複数行の移動も可能です。

関数の移動：command + shift + ↑/↓

タブの移動：command + shift + { / }

ウインドウの移動：option + (shift) + tab

shiftなしで時計回り、shiftありで反時計回りでウインドウがアクティブになります。

選択した対象の差分を表示：対象を選択して command + D

カンマ区切りの要素を交換：option + shift + command + →/←

下の行を持ってきて一行にする：control + shift + J

コンテキストアクション

ショートカットとはちょっと違いますが、option + enter を入力することでIDEが状況に応じた修正案を提示してくれます。
注）提示されるがままに実行すると謎コードになるぞ！使われるのでなく使いこなそう！

インターフェイスに準拠するメンバーの実装：option + enter

存在しないクラス等の作成：option + enter
